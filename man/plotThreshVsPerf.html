plotThreshVsPerf {mlr}R Documentation

Plot threshold vs. performance(s) for 2-class classification.

Description

Plot threshold vs. performance(s) for 2-class classification.

Usage

plotThreshVsPerf(pred, measures, mark.th = NA_real_, gridsize = 100L,
  linesize = 1.5)

Arguments

pred

[Prediction]
Prediction object.

measures

[Measure | list of Measure]
Performance measure(s) to evaluate.

mark.th

[numeric(1)]
Mark given threshold with vertical line? Default is NA which means not to do it.

gridsize

[integer(1)]
Grid resolution for x-axis (threshold). Default is 100.

linesize

[numeric(1)]
Linesize for ggplot2 geom_line for performance graphs. Default is 1.5.

Value

ggplot2 plot object.
