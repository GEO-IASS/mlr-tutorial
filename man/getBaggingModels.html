getBaggingModels {mlr}R Documentation

Returns the list of models fitted in bagging.

Description

Returns the list of models fitted in bagging.

Usage

getBaggingModels(model, learner.models = FALSE)

Arguments

model

[WrappedModel]
Model produced by training a bagging learner.

learner.models

[logical(1)]
Return underlying R models (e.g., rpart models) or wrapped mlr models (WrappedModel). Default is FALSE.

Value

[list].
